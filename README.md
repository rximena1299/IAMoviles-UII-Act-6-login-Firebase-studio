# IAMoviles UII Act 6  login Firebase studio
Desarrolla una aplicación móvil en Flutter que implemente un sistema de autenticación (Login y Sign Up) inspirado en la interfaz de Apple Music, utilizando un diseño moderno, limpio y minimalista.

La aplicación debe estar estructurada en múltiples archivos para mantener una arquitectura organizada y profesional. La estructura del proyecto debe incluir:

* main.dart como punto de entrada
* Una carpeta "screens" que contenga:

  * welcome_screen.dart
  * login_screen.dart
  * signup_screen.dart
* Una carpeta "widgets" que contenga:

  * components.dart (para reutilizar botones, campos de texto y estilos)

El diseño visual debe seguir la estética de Apple Music, utilizando una paleta de colores clara. El fondo debe ser un degradado vertical que combine tonos rosados suaves (#FFE5EC) con blanco, evitando fondos oscuros.

La pantalla principal (WelcomeScreen) debe incluir:

* El logo de la aplicación cargado desde GitHub (archivo llamado "logo.png")
* Un título "Apple Music"
* Un subtítulo descriptivo
* Dos botones principales: "Login" y "Sign Up"
* Una firma en la parte inferior con el texto "Ximena Reyes 6º I"

La navegación entre pantallas debe implementarse mediante Navigator.

La pantalla de Login debe contener:

* Logo en la parte superior
* Campos de texto para Email y Password
* Un botón principal "Entrar" con color rojo característico (#FF2D55)
* Un botón o texto para redirigir al registro

La pantalla de Sign Up debe incluir:

* Logo
* Campos para Username, Email, Password y Confirm Password
* Botón principal "Registrarse"

Los componentes reutilizables deben incluir:

* Botones principales (fondo rojo con texto blanco)
* Botones secundarios (borde rojo con fondo blanco)
* Campos de texto con estilo limpio y bordes redondeados

El diseño debe ser responsivo, con buen espaciado, bordes redondeados y jerarquía visual clara. Se deben evitar estilos recargados, priorizando una apariencia elegante y consistente con Apple Music.

La aplicación debe estar lista para ejecutarse en Flutter y poder visualizarse correctamente en DartPad en su versión inicial (main.dart).
